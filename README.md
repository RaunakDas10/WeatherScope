# WeatherScope

A stunning, fully responsive, cinematic weather dashboard built with React, Vite, and Tailwind CSS. 

[**Live Demo**](https://weather-scope-rho.vercel.app/)

WeatherScope visualizes real-time weather data with dynamic glassmorphism aesthetics, utilizing the OpenWeatherMap API for accurate metrics and the Google Gemini Imagen API (with a Pollinations AI fallback) to generate beautiful, contextual AI backgrounds that match the weather conditions of the selected city.

## Features

- **Cinematic Weather Aesthetics:** Enjoy a highly immersive visual experience with a storm-themed background, animated rain grain overlays, and glassmorphism panels.
- **Dynamic AI Backgrounds:** The background image intelligently changes based on the searched city and current weather condition, generated by state-of-the-art AI.
- **Real-Time Data:** View accurate current temperatures, high/lows, rain chances, wind speed, and humidity.
- **Responsive Design:** Fully fluid UI that scales beautifully on both desktop and mobile devices.
- **7-Day Forecast Strip:** Browse an interactive horizontal forecast strip that includes a beautiful SVG wave chart mapping temperature variations.
- **Air Quality Indicators:** Color-coded pollution markers showing the air quality index dynamically.

## Tech Stack

- **Frontend Framework:** React (Vite)
- **Styling:** Tailwind CSS + custom CSS modules
- **Animations:** Framer Motion
- **APIs Used:**
  - OpenWeatherMap API (Current weather, 7-day forecast, air quality)
  - Google Gemini Imagen 4.0 API (Dynamic backgrounds)
  - Pollinations AI (Seamless instant AI background fallback)

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/RaunakDas10/WeatherScope.git
   cd WeatherScope
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Setup:**
   Create a `.env` file in the root of the project and add your API keys:
   ```env
   VITE_OPENWEATHER_API_KEY=your_openweathermap_api_key_here
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```
   *(Note: The Gemini API key is optional as the app has a built-in free fallback using Pollinations AI!)*

4. **Start the development server:**
   ```bash
   npm run dev
   ```
